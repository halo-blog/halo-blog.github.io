Junit单元测试

测试分类：

  1. 黑盒测试：指定输入看输出
  2. 白盒测试：关注程序执行流程

Junit属于白盒测试

  1. 定义一个测试类(测试用例)

    测试类名：被测试的类名Test

    包名：xxx.xxx.xx.test

  2. 定义测试方法：可以独立运行

    方法名：test测试的方法名

    返回值：void

    参数列表：空参

  3. 给方法加上@Test，导入Junit依赖环境

@Before

初始化方法：用于资源申请，所有测试方法在执行之前都会执行该方法

@After

释放资源方法：在所有测试方法执行完后，都会自动执行该方法
